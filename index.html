Configure Network

  • IP Address: 203.100.x.1

  • Gateway: kosong

  • Username & Password: bebas

Networking Configuration

Tips Navigasi Nano Editor

Ctrl + O Save file (write out)
Ctrl + X Exit nano
Ctrl + W Search dalam file
Ctrl + K Cut line
Ctrl + U Paste line
1

Edit Network Interfaces

Jalankan command berikut:

nano /etc/network/interfaces

Isi file dengan konfigurasi berikut:

allow-hotplug enp0s3 iface enp0s3 inet dhcp auto enp0s8 iface enp0s8 inet static address 203.100.x.1 netmask 255.255.255.0
Penting: Konfigurasi di atas WAJIB sama persis. Perhatikan indentasi pada bagian address dan netmask.
Networking Done. Lanjut ke DHCP Server.

DHCP Server Setup

1

Install DHCP Server

Masukkan Disk 2, kemudian jalankan:

apt-cdrom add apt install isc-dhcp-server
2

Configure DHCP

Edit file konfigurasi DHCP:

nano /etc/dhcp/dhcpd.conf

Search & Replace di Nano

Gunakan Ctrl + \ untuk membuka fitur search & replace:

  • Search for: 10.5.5
  • Replace with: 203.100.x
  • Tekan a untuk replace all

Struktur File dhcpd.conf (Bagian Subnet)

File ini berisi konfigurasi DHCP. Cari section yang dimulai dengan "subnet":

~52 # A slightly different configuration for an internal subnet.
53 subnet 203.100.x.0 netmask 255.255.255.0 {
54 range 203.100.x.5 203.100.x.254;
55 option domain-name-servers 8.8.8.8,203.100.x.1;
56 option domain-name "smkn3.co.id";
57 option routers 203.100.x.1;
58 option broadcast-address 203.100.x.255;
59 default-lease-time 600;
60 max-lease-time 7200;
61 }

Edit baris-baris berikut sesuai dengan referensi di atas:

Line 53: netmask ubah dari 255.255.255.224 → 255.255.255.0
Line 54: range ubah dari 26-30 → 5-254
Line 55: hapus text setelah "servers" sampai "nfs" → ganti dengan 8.8.8.8,203.100.x.1
Line 56: domain name dalam tanda kutip → "smkn3.co.id"
Line 57: tidak ada yang diubah
Line 58: broadcast address dari .33 → .255
Tips: Gunakan Ctrl + W dan ketik "subnet" untuk menemukan section ini dengan cepat. Line number mungkin sedikit berbeda, yang penting cari section "subnet" yang sesuai.
3

Set Interface DHCP

nano /etc/default/isc-dhcp-server

Cari baris yang berisi INTERFACESv4 dan ubah menjadi:

INTERFACESv4="enp0s8"
Catatan: Gunakan Ctrl + W dan search "INTERFACE" untuk menemukan baris ini dengan cepat.
DHCP Server Done. Lanjut ke Firewall Configuration.

Firewall Configuration

1

Install iptables

apt install iptables-persistent

Saat ditanya save rules, pilih Yes.

2

Configure NAT & Masquerade

netfilter-persistent iptables -t nat -A POSTROUTING -o enp0s3 -j MASQUERADE netfilter-persistent save
Penjelasan: Command ini mengaktifkan NAT (Network Address Translation) sehingga client dapat mengakses internet melalui server.
3

Enable IP Forwarding

nano /etc/sysctl.conf

Cari baris berikut (gunakan Ctrl + W dan search "ip_forward"):

#net.ipv4.ip_forward=1

Hapus tanda # di awal baris sehingga menjadi:

net.ipv4.ip_forward=1
Firewall Done. Lanjut ke SSH Configuration.

SSH Server Setup

1

Configure SSH

nano /etc/ssh/sshd_config

Cari dan edit baris berikut:

Yang perlu diubah:
  • Cari "#Port 22" → Hapus tanda #
  • Cari "#PermitRootLogin" → Hapus tanda # dan ubah value-nya

Dari:

#PermitRootLogin prohibit-password

Menjadi:

PermitRootLogin yes
Restart Required: Jalankan command init 6 untuk restart server.

Testing & Troubleshooting

Perhatian: Bagian ini memerlukan ketelitian. Pastikan mengikuti setiap langkah dengan cermat.
1

Verifikasi IP dari DHCP

Dari client/laptop Windows, buka Command Prompt dan jalankan:

ipconfig
Expected Result:
  • IP Address harus berada di range 203.100.x.5 - 203.100.x.254
  • Default Gateway harus 203.100.x.1
2

Test Konektivitas

Lakukan ping test ke Google DNS:

ping 8.8.8.8
Success Indicator: Jika mendapat reply dari 8.8.8.8, konfigurasi berhasil.
3

Troubleshooting

Jika Gagal: Lakukan troubleshooting mandiri dengan mengecek kembali setiap konfigurasi yang telah dibuat.
Checklist Wajib:
  • Client harus mendapat IP dari server DHCP
  • Tidak perlu mengubah konfigurasi di Control Panel Windows
  • Pastikan semua service sudah berjalan di server
Konfigurasi Selesai. Semua konfigurasi di set oleh Jericho.
